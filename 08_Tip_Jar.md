# ТЗ: Tip Jar / Донат-страница на Bitcoin

## Обзор проекта

Генератор персональных страниц для приёма BTC-донатов. Пользователь создаёт страницу за 1 минуту, получает ссылку, делится с аудиторией. Донатеры оставляют BTC + сообщение. Лидерборд топ-донатеров.

---

## Целевая аудитория

- Стримеры, блогеры, контент-мейкеры
- Open-source разработчики
- Благотворительные проекты
- Любой, кто хочет принимать BTC-донаты

---

## Функциональные требования

### 1. Регистрация / Создание страницы

- Подключение Bitcoin-кошелька (Unisat, Xverse)
- Или ввод BTC-адреса вручную (без кошелька)
- Настройка страницы:
  - Имя / никнейм
  - Аватар (загрузка или URL)
  - Описание / био
  - Цель сбора (опционально, в BTC)
  - Пресет-суммы для быстрых донатов (100, 500, 1000 sats)
  - Тема оформления (3-5 тем)

### 2. Публичная донат-страница

- URL: /tip/:username
- Отображение: аватар, имя, описание
- Прогресс-бар к цели (если задана)
- Форма доната:
  - Сумма (пресеты + кастомная)
  - Сообщение (до 280 символов, опционально)
  - Имя донатера (опционально, анонимно по умолчанию)
- Оплата через QR-код или подключение кошелька
- Подтверждение: "Спасибо!" экран

### 3. Лидерборд

- Топ донатеров (за всё время / месяц / неделю)
- Отображение: ник (или "Anonymous"), сумма
- Бейджи: "Top Supporter", "Early Supporter"
- Общая сумма собранных донатов

### 4. Лента сообщений

- Публичная лента донатов с сообщениями
- Хронологический порядок
- Возможность скрыть сообщение (для владельца)

### 5. Дашборд владельца

- Общая статистика: собрано, количество донатов, уникальных донатеров
- График донатов по дням
- Список всех донатов с деталями
- Настройки страницы
- Виджет для встраивания (iframe/link)

## Механика доната

```
1. Донатер открывает /tip/username
2. Вводит сумму и сообщение
3. Получает BTC-адрес получателя + QR-код
4. Отправляет BTC через любой кошелёк
5. Backend мониторит mempool.space:
   GET https://mempool.space/api/address/{address}/txs
6. При обнаружении транзакции → запись доната
7. При подтверждении → обновление лидерборда

Альтернатива: Lightning Network
  - LNURL или Lightning invoice
  - Мгновенное подтверждение
  - Меньшие комиссии для мелких донатов
```

## Структура страниц

```
/ — Лендинг: "Создай свою страницу за 1 минуту"
/create — Создание/настройка страницы
/tip/:username — Публичная донат-страница
/dashboard — Дашборд владельца
/leaderboard/:username — Лидерборд страницы
```

## API эндпоинты

| Метод | Путь | Описание |
|-------|------|----------|
| POST | /api/pages | Создать страницу |
| GET | /api/pages/:username | Данные страницы |
| PUT | /api/pages/:username | Обновить настройки |
| POST | /api/donate/:username | Зарегистрировать донат |
| GET | /api/donations/:username | Список донатов |
| GET | /api/leaderboard/:username | Лидерборд |
| GET | /api/stats/:username | Статистика |

## MVP-скоуп

1. Создание страницы (имя + адрес + описание)
2. Публичная страница с QR-кодом
3. Отслеживание входящих транзакций
4. Лента донатов с сообщениями
5. Простой лидерборд

---

## Что НЕ входит в MVP

- Lightning Network интеграция
- Кастомные темы оформления
- OBS-виджет для стримов
- Recurring донаты (подписки)
- Мультивалютность
- Верификация личности

## Риски и митигация

| Риск | Митигация |
|------|-----------|
| Не видим транзакцию | Polling mempool.space каждые 30 сек + ручная кнопка "Check" |
| Спам-сообщения | Минимальная сумма доната для сообщения |
| Abuse страниц | Модерация + report кнопка |
| Privacy донатеров | Анонимность по умолчанию |
